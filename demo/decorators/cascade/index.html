PFE Cascade Decorator

A LitElement decorator which cascades a given property to specified child elements./p>

The <cascade-example> below are simple wrapper elements containing a Shadow DOM <pfe-band>.

The color attribute on the <cascade-example> is cascaded to the <pfe-band> Shadow DOM child.

    @cascades('pfe-band')
    @property({ reflect: true }) color: 'accent'|'base'|'complement'|'darker'|'darkest'|'lightest' = 'base';

Example with accent set as color property

The cascade does not respect direct child elements that have preset property values in the Light DOM and they will be overwritten, unless there is another decorated element wrapping the child as shown by the following two demos.
